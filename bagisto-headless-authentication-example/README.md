# Bagisto Headless eCommerce Authentication System Integration

[![Bagisto Headless](https://img.shields.io/badge/Bagisto-Headless-blue.svg)](https://bagisto.com/en/headless-ecommerce/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A professional, high-performance authentication starter kit built for **Bagisto Headless eCommerce**. This project demonstrates best practices for implementing secure, robust authentication using **NextAuth.js** and **GraphQL** with the Bagisto backend.

## üõ†Ô∏è Getting Started

### Prerequisites

Ensure you have a Bagisto backend installed and the [Bagisto GraphQL API](https://api-docs.bagisto.com/api/introduction.html) configured.

### Installation

1.  **Clone the project**:
    ```bash
    git clone https://github.com/bagisto/bagisto-auth-test.git
    cd bagisto-auth-test
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Environment Setup**:
    Create a `.env` file based on the provided `.env.example` and configure your Bagisto GraphQL endpoint and NextAuth secret.

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

5.  **Access the application**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.
